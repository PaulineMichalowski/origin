Gestion de documents

Tout d'abord, il est important de se rappeler que presque toutes les commandes possèdent un manuel, que l'on peut consulter à tout moment pour se rappeler les options ou la fonction de la commande, grâce à la commande man. Par exemple man ls ouvrira le manuel de la commande ls et ses options.

Pour créer un dossier je dois utiliser cette commande : mkdir + 'nom du dossier'. Je peux ensuite aller dans ce nouveau dossier avec la touche cd + 'nom du dossier'. Pour revenir en arrière, j'utilise cd .., qui me ramène sur le dossier précédent.

Pour créer un fichier, j'utilise : touch + 'nom du fichier'.

Je peux vérifier où je me situe dans les dossiers, avec la commande pwd. Je peux également vérifier quels dossiers et fichiers contient le dossier où je me trouve avec la commande ls.

Je peux utiliser la touche tabulation pour finir d'écrire un nom de dossier ou de fichier.

Je peux ouvrir un fichier sous n'importe quel logiciel avec la commande : 'nom du logiciel' + 'nom du fichier'

Utilisation du sudo, root, super-utilisateur

Je peux passer en "super-utilisateur", en utilisant la commande sudo -s (pour y rester) ou juste sudo, juste le temps d'une ligne de commande. Il faut être particulièrement vigilant, l'utilisation du super-utilisateur peut avoir des conséquences sur le système s'il n'est pas bien utilisé

En super-utilisateur, la commande apt update met à jour la liste des fichiers disponibles dans les dépôts APT présents dans le fichier de configuration.
La commande apt upgrade, quant à elle, met à jour les dernières versions de tous les paquets installés sur le système.

La commande dpkg -i + 'nom du paquet.deb' installe le ou les paquets indiqués, au format .deb. Cette commande s'utilise si le paquet n'est pas déjà dans les dépôts d'Ubuntu. Si le paquet est dans les dépôts, on utilise alors la commande apt install + 'nom du paquet'.
